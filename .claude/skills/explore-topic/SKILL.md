# Explore Topic

Deep multi-agent exploration of a research question or design challenge.

## Usage

```
/explore-topic "How should episodic memory consolidation work?"
/explore-topic "Type 1/Type 2 arbitration algorithm design"
/explore-topic "Drive cross-modulation strategies"
```

## When to Use

- When a design question needs multiple perspectives
- When researching approaches before planning an epic
- When Jim wants to think through a problem collaboratively

---

## Workflow

### Phase 1: FRAME

1. Identify the question or challenge
2. Determine which agents have relevant perspectives
3. Define the exploration scope

### Phase 2: PARALLEL EXPLORATION

Launch relevant agents in parallel:
- Science agents explore theoretical grounding
- Technical agents explore implementation feasibility
- Both search for relevant research, patterns, and prior art

### Phase 3: SYNTHESIS

1. Collect agent perspectives
2. Identify areas of agreement and tension
3. Present options with tradeoffs to Jim

---

## Output

```
docs/explorations/YYYY-MM-DD-slug.md
```

Contains: question framing, agent perspectives, synthesis, and recommended next steps.
